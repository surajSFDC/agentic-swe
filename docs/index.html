// how it works

Two paths, one pipeline

Every task is analyzed and routed. Simple fixes take the fast path. Complex features get full design review, testing, and multi-agent code review.

Fast path — simple tasks
feasibility → fast-path-check → fast-implementation → validation → pr-created → approval-wait → completed
Full path — complex tasks
feasibility → design → design-review → verification → test → implementation → self-review → code-review → permissions → validation → pr-created → approval-wait → completed

■ Human gates stop the pipeline for your review. Nothing ships without approval.

// capabilities

What makes it different

⚙

Zero runtime code

The entire pipeline is pure markdown. Claude Code reads the policies and executes the workflow. No servers, no dependencies, no build step.

◆

Evidence-based decisions

Every phase output follows a four-point evidence standard: observed, inferred, evidence, uncertain. No guessing.

★

135+ specialized agents

Auto-selected based on your codebase. Python, TypeScript, Rust, Kubernetes, security, ML — agents that know the domain.

☍

Human gates built in

The pipeline stops at ambiguity, approval, and escalation points. You stay in control of what ships.

⚙

Multi-agent review

Complex designs get a three-agent panel review: architect, security, and adversarial — running in parallel.

⚡

Fast path routing

Simple tasks skip design review and go straight to implementation. The pipeline adapts to task complexity automatically.

Up and running in 3 steps

1

Install into your project

npx agentic-swe /path/to/your/project
2

Open Claude Code

cd /path/to/your/project && claude
3

Start working

/work Add retry logic to the API client
// commands

Key commands

/work <task>Start a new task — auto-routes fast or full path
/work <id>Resume paused work by ID
/plan-only <task>Analyze and design without implementing
/subagentBrowse 135+ specialized subagents
/repo-scanStructured codebase snapshot
/check budgetVerify iteration and cost budgets
