# Agentic SWE Pro and services

The core pipeline is **MIT-licensed** open source. Paid offers are **people, priority, and packaging** around that same artifact—not a different executable or a cloud runtime you deploy from this repo.

## Boundary (read this first)

| Included in core (MIT) | Pro / services (commercial) |
|------------------------|----------------------------|
| Plugin-root pipeline (`commands/`, `phases/`, `agents/`, …), Claude Code plugin | Time-boxed help: install, tune phases, train teams |
| Runs in **your** Claude Code + **your** repo | Email/office hours / early add-on **content** (when offered) |
| You operate upgrades via **plugin marketplace** / **`claude --plugin-dir`** | No obligation to buy Pro to use the OSS pipeline |

**Still not sold:** execution of your code on our servers, a proprietary “Pro runtime,” or an API that runs tasks without your environment.

## What we do not sell (today)

- **Hosted SaaS runtime** — The pipeline runs in **Claude Code** on the customer’s machine; there is no multi-tenant cloud product in this repo.  
- **Guaranteed SLAs** — Unless agreed in a signed enterprise statement of work.

## First paid offers

### 1. Implementation and onboarding (services)

Fixed-scope engagements, for example:

- Install Agentic SWE into your org’s repos and tune `CLAUDE.md` / phases for your SDLC  
- Train the team on `/work`, budgets, gates, and artifacts  
- Optional customization of phase prompts and subagent selection for your stack  

**Best for:** Teams adopting Claude Code who want a guided rollout instead of self-serve only.

### 2. Agentic SWE Pro (lightweight bundle)

A **subscription-style** offer (details and pricing to be finalized) may include:

- **Email support** for installation and pipeline usage questions  
- **Office hours** or periodic Q&A (format TBD)  
- **Early access** to optional add-on packs (e.g. vertical subagent bundles) when available  

**Not included:** Hosted execution of your code or a managed API.

### 3. Training (optional)

Cohort or workshop on the **method** (state machine, evidence artifacts, human gates) using this repo as the reference implementation.
