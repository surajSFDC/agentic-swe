# Capability gaps (section template)

Paste into `implementation.md` (or summarize in validation) when the catalog does not cover required expertise.

```markdown
## Capability gaps

- **Gap**: <what expertise or tooling was needed but missing>
- **Evidence**: <task or files that exposed the gap>
- **Mitigation this run**: <what you did instead, or scope reduction>
- **Follow-up (optional)**: Add a repo-specific specialist under `${CLAUDE_PLUGIN_ROOT}/agents/subagents/custom/<name>.md` (see project usage docs), or extend org docs in `docs/agentic-swe/`.
```

Omit the section entirely if no gaps.
